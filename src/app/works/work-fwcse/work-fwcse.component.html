WordPress & CSS Frontend

This website was a project we worked on back in 2012 and as it stands on its own, it may not be visually too impressive, but there is a story behind this project that helped shape what Orange Circle Company was to become.

This WordPress theme was developed on-top of an existing theme called Happy Cyclope

Back in 2012 Javascript and actual programming was still very new to us, and so we would try to solve all problems using only HTML and CSS.

The Ficcadenti Waggoner and Castle website as it exists today was literally built without changing and PHP or Javascript Code. It is the exact same code as the Happy Cyclope theme, but with modified CSS.

To put this in perspective this is the exact same thing that CSS Zen Garden illustrates on their website.

The reason we feature this work as a former client is to show both how much we have grown skill wise, as well as the problem solving skills and creativity used in any project we tackle.
