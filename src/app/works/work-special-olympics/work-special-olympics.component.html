The Special Olympics GMS project is something we feel very fortunate to be part of. It was an AngularJS frontend that connected with a Microsoft build API for gathering and organizing athlete data.

The application allows users to select a Delegation or select a Sport, and then from there view athletes results.

Our Role

We worked on 2 aspects of the project.

The Artwork

We prepared the dozens of Adobe illustrator vector files and converted them into web ready images for each of the sports

The Frontend

We also helped build the Frontend of the application. It is built with Uikit and custom CSS that we developed based on the designs of some talented designers.
